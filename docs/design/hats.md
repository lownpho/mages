# Hats

Three hat families with 5 tiers each. Hats occupy the hat equipment slot.
Hats are the offensive slot — every hat gives skill (damage scaling). The choice is what
defensive stat, if any, you get alongside it.

---

## Hat

Pure skill. No secondary stats. The glass cannon option.

| Tier | +HP | +Mana | +Skill | +Speed |
|------|-----|-------|--------|--------|
| Hat | 0 | 0 | 10 | 0 |
| Nice Hat | 0 | 0 | 20 | 0 |
| Nicer Hat | 0 | 0 | 40 | 0 |
| Fancy Hat | 0 | 0 | 80 | 0 |
| Ultimate Hat | 0 | 0 | 150 | 0 |
